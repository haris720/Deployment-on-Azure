# My Treats Platform

A full-stack restaurant discovery and reservation platform.

## Technology Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- Node.js
- Express
- TypeScript
- Prisma

### Database
- PostgreSQL

### Deployment
- Frontend: Azure App Service
- Backend: Azure Linux Virtual Machine
- CI/CD: GitHub Actions
